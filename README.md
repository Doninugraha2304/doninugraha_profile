# Doni Nugraha - Portfolio Website

A modern, minimalist portfolio website built with Next.js 14, TypeScript, Tailwind CSS, and Shadcn UI.

## 🚀 Features

- **Modern Stack**: Built with Next.js 14, TypeScript, and Tailwind CSS
- **Beautiful UI**: Minimalist design using Shadcn UI components
- **Responsive**: Fully responsive design that works on all devices
- **Dark Mode**: Automatic dark mode support based on system preferences
- **Fast**: Optimized for performance with Next.js App Router
- **Icons**: Lucide React icons for a modern look

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Shadcn UI
- **Icons**: Lucide React
- **Font**: Geist Sans & Geist Mono

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Doninugraha2304/doninugraha_profile.git

# Navigate to the project directory
cd doninugraha_profile

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Build for Production

```bash
npm run build
npm start
```

## 📝 Customization

To customize the portfolio with your own information:

1. Edit `app/page.tsx` to update skills, experiences, and projects
2. Edit `app/layout.tsx` to update metadata (title, description)
3. Modify colors and styling in `app/globals.css`
